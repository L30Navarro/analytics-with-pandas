# analytics-with-pandas
Repositório dedicado à análise de dados, utilizando a biblioteca Pandas em Python. Inclui exemplos práticos de limpeza, transformação, exploração e visualização de dados aplicados a diferentes cenários de _analytics_.

## Status do projeto
Este repositório está em desenvolvimento.
Atualmente contém as bases de dados iniciais que serão utilizadas
para análises exploratórias com Pandas.
